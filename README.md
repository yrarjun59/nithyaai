# 🌸 NithyaAI — नित्या

> *Nithya (नित्या) — Sanskrit for "eternal, always present"*

NithyaAI is a locally-running AI companion that understands and responds in modern conversational Nepali.
Built on top of Ollama and open-source LLMs, Nithya runs entirely on your machine — your conversations stay yours.
